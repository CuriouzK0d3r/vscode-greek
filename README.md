# Greek Language Pack for VS Code

This extension provides Greek translations for the Visual Studio Code user interface.

## How to use

1. Install this extension.
2. Press `Ctrl+Shift+P` to open the Command Palette.
3. Type "Configure Display Language" and press Enter.
4. Select "Greek (Ελληνικά)" from the list.
5. Restart VS Code.

## Contribution

This is a basic implementation. For more translations, please contribute to the `translations/main.i18n.json` file.
